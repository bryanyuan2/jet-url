# JetURL

**Mozilla Jetpack 50-line Code Challenge**

A lightweight bookmark management Firefox extension built with Mozilla Jetpack API. This tool allows users to quickly save and manage web pages through an intuitive sidebar interface with seamless access to bookmarked content.
